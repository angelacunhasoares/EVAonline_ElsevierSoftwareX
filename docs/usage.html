Usage Guide

This guide explains how to use EVAonline to calculate reference evapotranspiration (ET₀) using the FAO-56 Penman-Monteith model. Start on the Home page to explore maps and select coordinates, then proceed to the Calculate ETo page to perform calculations and view results.

Step-by-Step Instructions

Home Page

  1. Select a Map

    Choose a map to visualize geographic data or select coordinates:

    • All Cities in MATOPIBA: Displays all municipalities in the MATOPIBA region with a perimeter outline.
    • Piracicaba City: Shows the city of Piracicaba with INMET station coordinates.
    • INMET Stations: Displays INMET weather stations with filters for automatic or conventional stations.
    • Global Map: Allows clicking to select coordinates anywhere in the world.

  2. Filter INMET Stations (if selected)

    If you choose the INMET Stations map, use checkboxes to filter:

    • Show All Stations: Display all INMET stations.
    • Show Automatic Stations: Display only automatic stations.
    • Show Conventional Stations: Display only conventional stations.
  3. Capture Coordinates (Global Map)

    Click on the Global Map to capture coordinates (latitude and longitude), then click Calculate ET₀ to proceed to the Calculate ETo page.

  4. Explore Additional Options

    Use buttons to:

    • About: Learn more about EVAonline.
    • Read the Docs: Access this documentation.
    • Switch Language: Toggle between languages (e.g., English, Portuguese).

Calculate ETo Page

  1. Choose Calculation Mode

    Select the calculation mode from the dropdown menu:

    • Global: Calculate ET₀ using coordinates from the Home page map or manual input.
    • MATOPIBA: Calculate ET₀ for a specific city in the MATOPIBA region.

  2. Define Coordinates

    Depending on the mode, you have the following options:

    • Global Mode:
      • Use Coordinates from Home: Apply coordinates captured on the Home page.
      • Click on Map: Select coordinates on an interactive map.
      • Manual Input: Enter latitude (-90 to 90) and longitude (-180 to 180).

    • MATOPIBA Mode:
      • Select City: Choose a state and city from dropdown menus.
      • Manual Input: Enter coordinates within the MATOPIBA region (-14.5 to -2.5 lat, -50.0 to -41.5 long).

  3. Select Data Source and Period

    Choose a data source (NASA POWER, Open-Meteo Archive, Open-Meteo Forecast, INMET) and specify the start and end dates.

  4. Confirm Parameters

    Review the selected parameters (mode, coordinates, data source, dates, elevation) and click Confirm. Adjust the elevation if needed.

  5. Calculate ET₀

    Click the Calculate ET₀ button to generate results. Results are displayed in two tabs: Table and Graphs and Statistics.

  6. View Results

    Explore results in the following tabs:

    • Table and Graphs:
      • Table: Displays ET₀ and climatic variables (e.g., temperature, radiation, humidity).
      • Graphs: Choose from:
        • ET₀ vs. Temperature
        • ET₀ vs. Radiation
        • ET₀ vs. Temperature vs. Radiation vs. Precipitation
        • Heatmap of Correlations
        • Correlation Graph

    • Statistics:
      • Daily climatic data with ET₀
      • Descriptive statistics
      • Normality test
      • Correlation matrix
      • ET₀ summary
      • Cumulative distribution (optional)

  7. Additional Actions

    Use buttons to:

    • Enter New Data: Reset inputs and start over.
    • Clear All: Clear all selections and results.
    • Back to Home: Return to the Home page.

User Options

Home Page Options

  • Map Selection:
    • All Cities in MATOPIBA: Visualize all municipalities in the MATOPIBA region.
    • Piracicaba City: View Piracicaba with INMET station coordinates.
    • INMET Stations: Display INMET stations with filters.
    • Global Map: Select coordinates by clicking anywhere in the world.
  • INMET Station Filters (if INMET Stations map is selected):
    • Show All Stations
    • Show Automatic Stations
    • Show Conventional Stations
  • Navigation:
    • About: Access information about EVAonline.
    • Read the Docs: View this documentation.
    • Switch Language: Toggle between languages (e.g., English, Portuguese).
  • Coordinate Capture: Click on the Global Map to capture coordinates for ET₀ calculation.

Calculate ETo Page Options

  • Calculation Mode: Global or MATOPIBA.
  • Coordinate Selection:
    • Global: Use coordinates from Home, click on map, or manual input (latitude -90 to 90, longitude -180 to 180).
    • MATOPIBA: Select state/city or manual input within region bounds (-14.5 to -2.5 lat, -50.0 to -41.5 long).
  • Data Source: NASA POWER, Open-Meteo Archive, Open-Meteo Forecast, INMET.
  • Date Range: Select start and end dates for data retrieval.
  • Elevation Adjustment: Automatically retrieved or manually adjusted (-1000 to 9000 meters).
  • Result Visualization:
    • Table: Display ET₀ and climatic variables.
    • Graphs: Choose from ET₀ vs. Temperature, ET₀ vs. Radiation, Heatmap, etc.
    • Statistics: View daily data, descriptive stats, normality tests, and more.
  • Additional Actions:
    • Use Coordinates from Home
    • Enter New Data
    • Clear All
    • Back to Home
